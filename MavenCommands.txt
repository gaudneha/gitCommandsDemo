Basic Maven Commands

	mvn clean -> clean the project and removes all files generated by the privious build
	mvn compile -> compile source code of the project
	mvn test-complie -> compile the test code
	mvn test -> test for the project
	mvn package -> create JAR or WAR file for the project to convert it into a distributable format
	mvn install -> Deploys the packaged JAR/WAR to the local repository
